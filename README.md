# Personal Portfolio Website

This project is a personal portfolio website built using HTML, CSS, and JavaScript.

## Features

- Home, About, Projects, Resume, Contact sections
- Dark Mode toggle
- Scroll animations
- Chatbot for answering FAQ
- QR Code for quick access
- GitHub project links

## Chatbot Implementation

A simple JavaScript chatbot was created to answer frequently asked questions about:

- Skills
- Projects
- Experience
- Contact information

The chatbot uses keyword matching to return predefined responses.

## How to Use

1. Open index.html in a browser
2. Navigate through the sections
3. Use the chatbot in the bottom corner
4. Download the resume from the Resume section

## Projects

Links to GitHub repositories are available in the Projects section of the website.
